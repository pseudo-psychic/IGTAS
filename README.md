# TAS bepinex plugin of IGTAP

requires:

[BepInEx 5.4.23.5](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.23.5)

download plugin [here](https://github.com/pseudo-psychic/IGTAS/releases/)

## setup:
Simply run the game after bepinex is in the folder to generate the plugins folder

Put the plugin .dll in the plugins folder

(tested on Mar 1, 2026)

## bugs:
Movement TAS is fixed physics-dependent, but jump height is time(framerate) dependent 

## use:
only uses keyboard

keybinds: \
f6 - start record \
f7 - stop recording or playback \
f8 - start playback \
\
f9 - open editor \
f10 - add frame \
f11 - remove frame \
left and right arrow - select frame in editor
